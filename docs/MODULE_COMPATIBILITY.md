# Compatibilidad de modulos

Estado inicial del bridge.

| Modulo | Estado | Notas |
| --- | --- | --- |
| Base | Parcial | Sesion, usuario, menus, acciones, campos, permisos, defaults, onchange y registros paginados. |
| Contacts | Parcial medido | `res.partner` genera IR `form,list`, lee campos y registros paginados. |
| Sales | No medido | Requiere instalar y probar modulo. |
| CRM | No medido | Requiere instalar y probar modulo. |
| Inventory | No medido | Requiere instalar y probar modulo. |

No se inventan porcentajes de compatibilidad hasta medir vistas reales.
